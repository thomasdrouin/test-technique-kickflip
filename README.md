# Kickflip technical test

This is a monorepo for the kickflip technical test.\
The `common` package contains the function and its test for the question 1.\
The `front` package contains a react app with the component for the question 2.

## Install

1. Have yarn 1 installed.
2. In project root, run `yarn`.

### Run function test for Question 1

In project root, run `yarn common test`.

### Run react app for Question 2

In project root, run `yarn fe start`.
